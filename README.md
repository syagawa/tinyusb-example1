# net example1

## install libraries

1. Start Arduino
2. Sketch -> Include Library -> Manage Libraries
3. search M5Stack and install M5Stack
4. search tinyusb and install ESP32TinyUSB and  Adafruit_TinyUSB_Arduino
5. select board M5Stack-ATOMS3
